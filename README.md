# 🖥️ Terminal Portfolio

Portafolio personal con estética de terminal inspirado en entornos Linux y herramientas CLI.  
Diseñado para presentar proyectos de forma clara, técnica y visualmente coherente.

---

## 🚀 Demo

🔗 [https://fiumgi.github.io](https://fiumgi.github.io)

---

## 📌 Características

- 🎨 Interfaz estilo terminal (tmux + neofetch)
- 🟣 Paleta personalizada basada en tonos morados
- 📊 Indicadores dinámicos (RAM, hora, uptime)
- 📜 Logs de sistema simulados
- 📁 Sección de proyectos tipo CLI
- 📱 Diseño responsive

---

## 🧠 Enfoque

Este portafolio no busca ser solo visual.  
Está diseñado para reflejar:

- Pensamiento estructurado  
- Interés por sistemas y terminales  
- Enfoque en herramientas reales (no solo UI)  

---

## 🛠️ Tecnologías

- HTML5  
- CSS3 (custom properties)  
- JavaScript (vanilla)

## 📦 Proyecto destacado

### 🍄 Fungi-Traductor

Traductor offline con interfaz gráfica basado en arquitectura MVC.

- Traducción sin conexión  
- Detección automática de idioma  
- Síntesis de voz (TTS)
